## [About](https://github.com/fosscord/fosscord-client/wiki)
The Fosscord client is the user interface to Fosscord servers and is backwards compatible to discord.com.

- Performant
- Open Source
- Themeable
- Extendable (Plugins)
- Secure
- Consistent (Looks the same on all platforms)
